Paper types

There are four paper types directly available in Origami Editor 3D:

  • Square
  • A4, which has an aspect ratio of 4243:3000
  • Regular hexagon
  • Dollar bill, which has an aspect ratio of 40:17

These can be found in the File > New submenu.

If you need another type of paper, you can either cut the paper to the size and shape you need with the Cutting Tool, or define a new paper type in the OrigamiScript Terminal. This can be done, for example, by typing paper [0 0 W H] new, where you have to replace W and H with the width and height of the paper in pixels.
