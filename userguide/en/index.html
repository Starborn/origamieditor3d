Origami Editor 3D User Guide

Thank you for downloading Origami Editor 3D.

Origami Editor 3D is a tool for creating complex origami models in a (mostly) realistic three-dimensional environment. Using it requires a basic understanding of geometry and some familiarity with origami techniques. You cannot learn origami from this program alone, though it can show you the step-by-step folding process of any origami model that was created in this editor.

Origami Editor 3D brings a novel element to computer origami modeling: the Reflection Tool, a versatile virtual folding method that can simulate reverse folds, squash folds, sink folds, petal folds and other types of folds with relative ease. Learning how to use this tool is the most important part of the learning curve.

This user guide is built up like a tutorial and has an intended reading order. You can also choose individual chapters from the table of contents below, but it is highly recommended to read through at least the Basic Editing part.
