Exporting origami to OpenCTM

Origami Editor 3D is capable of converting origami to a 3D graphics file format known as OpenCTM. This function can be accessed from the File > Export > To OpenCTM 3D File... menu. Of course, this will not save any of the folding process, only the finished model.

OpenCTM is an open triangle mesh format that is interchangeable with other 3D formats such as COLLADA, LightWave 3D or 3D Studio Max. An open source viewer and a conversion tool can be downloaded from the OpenCTM website.
