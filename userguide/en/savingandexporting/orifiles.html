What is an Origami File?

An origami file in Origami Editor 3D is not just the model you see in the 3D View. It is the chain of all the distinct folds made to the model, from start to finish. When you save an origami, the entire folding process is automatically saved with it. This ensures that none of your work is lost, and makes origami files a valuable resource for teaching, designing and analyzing origami figures.

To save an origami, select File > Save. Origami files are saved under the .ORI extension, in Origami Editor 3D's own file format. ORI files are usually very small: one folding command is stored in 16 bytes, then the file is compressed even further with an LZW algorithm. A typical intermediate level origami can be stored in around 200 bytes.
