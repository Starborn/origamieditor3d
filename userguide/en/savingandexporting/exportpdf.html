Exporting origami to PDF

When exporting to PDF, the program attempts to generate human-readable folding instructions from the data in the origami file, then saves it in a PDF file.

These folding instructions are usually very vague because Origami Editor 3D does not have the adequate AI to express the various use cases of Reflection Tool in human terms. Sometimes it can distinguish between a simple fold, a reverse fold and a sink fold, but it needs more development to be perfectly unambiguous at all times.
