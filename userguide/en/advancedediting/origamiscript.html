OrigamiScript Terminal

OrigamiScript is the internal scripting language of Origami Editor 3D. Every folding action or file operation in the GUI is first translated to OrigamiScript, then sent to the Terminal for execution.

You can access the Terminal directly by clicking on the Scripting tab. Commands can be entered into the bottom text field. You can retrieve a list of previously executed commands (including those that were sent from the GUI) with the log command.

You can also load OrigamiScript files by clicking File > Open... and choosing the OrigamiScript (*.txt) file type in the file dialog.

The documentation of OrigamiScript can be found here.
