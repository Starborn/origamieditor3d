Neusis Mode

Neusis Mode is a setting that allows the Ruler to be moved freely and be fixed at any position and angle with one click. It can be used for neusis constructions such as angle trisection. Of course, it can also be used as a less accurate replacement for the Angle bisector tool.

Neusis Mode can be switched on/off by clicking Edit > Neusis Mode. Entering Neusis Mode automatically enables Preview Mode because you need that for real neusis constructions.
